# Github Users

## 1. Deploy

Add `.env` file:

```env
LOGIN_USERNAME="login_username"
LOGIN_PASSWORD="login_password"
GITHUB_TOKEN="create_a_github_token"
NEXTAUTH_SECRET="create_a_nextauth_secret"
NEXTAUTH_URL="https://your.website.domain.com/"
```

Build project:

```bash
npm run build
```

Start project:

```bash
npm start
```
